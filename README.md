# MedNex Hospital Management System

MedNex Hospital Management System is a console-based Java application designed to manage hospital operations efficiently.

## Features
- Add, View, Update and Discharge Patients
- Manage Doctors and Departments
- Book and View Appointments
- Generate Unique Patient IDs
- Menu-driven console interface

## Technologies Used
- Java
- Object Oriented Programming (OOP)
- Collections Framework

## How to Run
1. Compile the project:
   javac src/*.java
2. Run:
   java src.MedNexHospital
